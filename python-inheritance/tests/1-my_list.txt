The ``1-my_list`` module
======================

Using ``print_sorted``
-------------------

Test for add_integer function.  First import
``print_sorted`` from the ``1-my_list`` module:

    >>> MyList = __import__('1-my_list').MyList

Now use it:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

    >>> new_list = MyList()
    >>> new_list.append(1)
    >>> new_list.append(-1)
    >>> my_list.print_sorted()
    [1, -1]

    >>> new_list = MyList()
    >>> my_list.print_sorted()
    []
